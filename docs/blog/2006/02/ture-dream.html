真实的幻梦

言吾 | Talk    南京 / 梦 / 诗歌
2006-02-06 09:06 @ Mon
全文共 173 字，已阅 次，需0分 47秒

原来作诗这么简单:em07:

闹钟时间指向两点三刻
我挣扎着
想拨开眼皮跳出来
但是没有用
手表的时针一样在向三爬去

忽然
闹铃声响起
时间是七点半
早晨

我一跃而起
该上班了…我想
虽然我还是无法确定

这一刻是否
比幻梦更真实

　—-2006年01月17日　于南京龙蟠客栈

• • •

在南京培训的一个清早，梦中醒来，脑子里就是这样一首所谓”Poem”，拿手机记录了下来，同时老脸绯红。
